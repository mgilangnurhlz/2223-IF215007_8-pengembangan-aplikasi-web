### Clone repository dan masuk folder backend
```sh
git clone https://github.com/mgilangnurhlz/2223-IF215007_8-pengembangan-aplikasi-web.git
```

### Install node module dan library
```sh
npm install
```

### Jalankan program
```sh
npm start
```
